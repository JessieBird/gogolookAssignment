# 人間清醒醫院 · Scam IQ Test 2.0

> **Stay-Woke Hospital** — 把「防詐測驗」包裝成急診室 RPG 的沉浸式體驗

為了解決傳統 Scam IQ Test 完測率低、無社交擴散力的痛點，把使用者從「答題者」翻轉為「急診醫師」，診斷 5 位真實詐騙情境中的病患，最終生出個人化的「防詐人格藥袋」在社群擴散，同時把 ScamAdviser 從「結算後的外部連結」變成「遊戲內主動使用的工具」。

---

## 🎯 核心假設

### 🟢 主假設（B2C Growth）

> 把傳統問答式 Scam IQ Test 改成**劇情驅動 RPG + 個人化身份診斷**，能讓**完測率提升 2 倍以上（目標 ≥ 50%）**、**分享率提升 3 倍（≥ 15%）**，以更低 CAC 為 ScamAdviser 導入 B2C 流量。

**為什麼敢這樣 claim？**
- RPG 化從「被考試的人」變成「幫別人看病的醫師」 → 消除心理防衛 → 完測率↑
- 個人化診斷 + 藥袋迷因創造社交貨幣（人人想曬自己拿到什麼藥） → 分享率↑
- 結算時推 CTA 正好是玩家情感高點（剛拿到段位） → 轉換率↑

### 🟡 輔助假設（Product Integration）

> 把 ScamAdviser 從「結算後的外部連結」變成「遊戲內的會診工具」，能在**情感高點**（玩家面對重症詐騙、答不出來時）讓使用者**主動使用它**，建立「ScamAdviser = 實用工具」而非「ScamAdviser = 廣告位」的產品印象，提高長期 retention。

## 🏥 產品機制設計

### 世界觀
- **場所**：人間清醒醫院・網路急診部
- **玩家身份**：剛到院的實習醫師（不是被考試者，而是診斷者）
- **任務**：為 5 位詐騙情境中的病患做急診診斷
- **初始資源**：❤️ 愛心 × 3　＋　✨ 洞察力 XP × 0

### Onboarding
- 輸入醫師姓名（會出現在診斷書 + 藥袋上，建立所有權）
- 選擇你遇到詐騙時的第一反應（4 種傾向：馬上查證 / 問別人意見 / 憑感覺判斷 / 先行動再想）
- 傾向不影響計分，僅顯示於診斷書增加代入感

### 關卡：5 位病患情境

| # | 病患類型 | 難度 | 決定性線索 | ScamAdviser 提示 |
|---|---|---|---|---|
| 1 | AI 變聲詐騙 | 🟠 重症 | 背景音罐頭警察對講機 | 刪除「先匯款救人」選項 |
| 2 | 假投資 LINE 群 | 一般 | 對帳單格式過度統一 | — |
| 3 | 假物流簡訊 | 一般 | `.vip` / `.xyz` 假網域 | — |
| 4 | 假求職陷阱 | 🟠 重症 | 保證金匯個人帳戶（非公司統編）| 刪除「先繳 3000 再說」選項 |
| 5 | 假公益募款 | 一般 | 無衛福部勸募字號 | — |

每題 4 個選項，依選項的 dim 總和（0~8）計分。

## 🎮 遊戲狀態系統（核心機制）

### ✨ XP 經驗值 + ❤️ 愛心生命
- **XP 公式**：`XP = 選項的 dim 總和 × 5`
  - 最佳選項（dim 總和 6~8）→ **+30~40 XP**
  - 中等選項（dim 總和 3~5）→ **+15~25 XP**
  - 最爛選項（dim 總和 = 0）→ **+0 XP，且扣 1 ❤️**
- **愛心**：初始 3 顆，累積選到最爛選項 3 次 → 歸零 → 觸發病危
- **選後回饋**：中央彈出浮動動畫（+XP or −1 ❤️）1.1 秒 → 再進下一題

### 🔍 ScamAdviser 會診（遊戲內產品整合 · 核心設計）
- **觸發條件**：只在重症題（Case 1, 4）顯示會診按鈕
- **成本**：25 XP（表示要累積 1~2 題才能用第一次）
- **XP 不足時**：按鈕灰色鎖定，標示「需 25 XP（目前 X）」形成「再撐一題就能用」的張力
- **啟用效果**：
  - 橘色會診報告欄位展開，揭露關鍵線索
  - 一個明顯錯誤選項被紅色「ScamAdviser 標記」標籤且灰掉不可點 → 4 選 1 降為 3 選 1
- **每題限用一次**

> 這是本 MVP 最關鍵的產品整合：把 ScamAdviser 從「外部廣告」升級為「玩家主動想用的道具」。

### 💀 病危機制（愛心歸零 → 復活 CTA）
- 愛心歸零時攔截遊戲流程，跳出 **Death Screen Modal**（Skull icon + CODE BLUE 警示）
- 兩個出口：
  - 🔴 **前往 Scam University 取得復活執照**（外連，retention 轉換點）
  - ⚪ 接受懲處，繼續完成剩餘病患（關 Modal 續玩）

## 🧩 6 種防詐人格矩陣

判定基於玩家累積的 4 維度：
`caution 警覺` / `rational 理性` / `skeptic 懷疑` / `independent 獨立`

| 人格 | 段位 | 判定條件（簡化）| 對應藥名 | 診斷章 |
|---|---|---|---|---|
| 冷血分析師 | 頂尖主任醫師 | total≥28 或 sp≥6+rp≥6 | 過度冷靜錠 500mg | 免疫出院 |
| 善良受難者 | 可憐實習生 | cp≤3 且 sp≤3 且 rp≤3 | 良心過剩膠囊 250mg | 需要住院 |
| 嘴砲專家 | 資深主治醫師 | rp≥7, sp=5, cp≤5 ⚠️ | 知行合一糖漿 100ml | 需觀察 |
| 直覺獵人 | 急診總醫師 | ip≥7, cp≥5, rp≤5 | 第六感維他命 50mg | 觀察中 |
| 制度信徒 | 住院醫師 | rp≥6, cp≥6, sp≤4 ⚠️ | 假官方辨識丸 200mg | 需補修 |
| 混沌中立者 | 謎之醫師 | 4 維度極差 ≤ 3 | 謎之安慰劑（適量 × 30 粒）| 特殊體質 |

⚠️ 標記者：現有題目資料下數學上難以達成。為 demo 展示，支援網址 `?p=<id>` 覆寫。

### 每個人格的個人化內容
不只是名字不同──處方箋 + 藥袋的**藥名、服用時機、副作用、醫師囑咐、CTA 出口**都不一樣，製造「啊這就是我」的辨識感。

**舉例：**
- **善良受難者** → 良心過剩膠囊 250mg，每日 4 次 + 睡前，飯前服用；其它用藥指示：「同情心發作時緊急加服 1 粒，搭配深呼吸十秒效果更佳。」
- **嘴砲專家** → 知行合一糖漿 100ml × 2 瓶；其它用藥指示：「點任何連結前 3 秒吞服；嚴禁『先點再說』，謝謝。」

## 🦠 擴散機制設計：為什麼是台灣藥袋？

作業要求「激發勝負慾或炫耀心理」。這裡**不做第二張 MBTI 風格人格卡**（已過度飽和），而選擇一個**具強文化辨識度 + 黑色幽默**的載體：**台灣診所藥袋**。

### 三層擴散動力
1. **視覺辨識度**：綠色框線 + 打勾方格 + 牛皮紙質感，台灣人一看就懂「這是藥袋」──不用解釋
2. **黑色幽默**：藥名本身就是笑點（過度冷靜錠 500mg × 終身劑量、良心過剩膠囊、謎之安慰劑 適量 × 30 粒）
3. **自嘲感 = 社交貨幣**：「我被診斷為善良受難者，拿了 30 粒良心過剩膠囊」比「我答對 3/5 題」更容易被 IG story 分享

### 技術實作
- `html-to-image` 把 DOM 轉成 PNG，支援一鍵下載
- Web Share API fallback 觸發手機原生分享面板（可直接帶檔案 + 預設文案）
- 檔名自動帶使用者姓名：`人間清醒醫院-{姓名}-{人格}藥袋.png`

## 📊 關鍵指標 · 6 層漏斗（以 1000 人入站為基準）

| # | 指標 | Go 門檻 | 預期數字 | 對應假設 |
|---|---|---|---|---|
| ① | 首頁 → Onboarding CTR | ≥ 70% | 700 | 主 |
| ② | **完測率（完成第 5 題）** ⭐ | **≥ 50%** | **500** | **主（最重要）** |
| ③ | 重症題會診使用率 | ≥ 30% | 150+ | 輔 |
| ④ | 擴散行為（下載 / 分享藥袋） | ≥ 15% | 75+ | 主 |
| ⑤ | K-factor（每完測者帶來的新訪客）| ≥ 0.3 | 150+ 新訪 | 主 |
| ⑥ | 處方箋 / 藥袋 → Scam Uni / ScamAdviser CTR | ≥ 10% | 50+ 外連 | 輔 |

### Go / No-Go 判準

- ✅ **Go 訊號**：② 完測率 ≥ 50% **AND** ④ 擴散率 ≥ 15%
- ❌ **No-Go 訊號**：② 完測率 < 35% → RPG 化反而提高門檻，需簡化機制或降低題數
- ⚠️ **模糊訊號**：
  - 完測率高但擴散率低 → 遊戲黏著 OK 但藥袋梗不夠強，迭代視覺文案
  - 擴散率高但完測率低 → 結算頁吸引但過程太長，縮減題數（5 → 3）
  - 會診使用率 < 10% → 產品整合位置不明顯或 XP 成本太高，重新調整門檻

- 1000 人樣本分 2 週導入，避開週日流量高低峰影響
- 建議同時跑**原版 Scam IQ Test** 做 A/B test 對照組
- 衡量基準：傳統 quiz 型測驗的完測率業界基準為 20~35%

## 🎤 面試論述重點（talking points）

### 1 句話定位
> 「把防詐測驗從 quiz 變 RPG，用醫療隱喻 + 台灣藥袋迷因做成可擴散的社交貨幣，同時把 ScamAdviser 從外部連結變成遊戲內主動使用的工具。」

### 3 個最能講的設計決策

**① 從「被測驗者」翻轉成「診斷者」**
傳統 Scam IQ Test 讓玩家覺得被質疑智商；RPG 讓玩家變醫師、幫別人看病 → 無心理防衛 + 情感代入 → 完測率直接提升。

**② ScamAdviser 從「廣告位」升級成「遊戲機制」**
會診按鈕出現在玩家最想用它的時刻（重症題卡關），用 XP 消耗建立「它有價值」的認知。這比結算後才推 CTA 深一層。

**③ 藥袋做擴散載體（vs 通用人格卡）**
台灣藥袋有強文化符號 + 黑色幽默 + 社群 context → 比「我是 XX 人格」這種通用圖卡更容易觸發分享。

### 可預期被 challenge 的問題 & 回答

- **Q：為什麼敢 claim 完測率 50%？**
  A：傳統 quiz 基準 20-35%，我們做了 3 件關鍵事——RPG 化消除答題疲勞、每題 20-30 秒控制在 3-5 分鐘完測時間、情感高點（愛心系統）讓人想撐完；即使有「愛心歸零想放棄」的流失風險已計入，50% 是可 defend 的積極目標。

- **Q：藥袋為什麼不做 MBTI 風格人格卡？**
  A：MBTI 圖卡已過度飽和；台灣藥袋有強在地文化 context（一看就懂、一看就笑），黑色幽默比資訊美感更能驅動「我想分享」的衝動。

- **Q：遊戲內的 ScamAdviser 會診會不會 cannibalize 本站流量？**
  A：相反。玩家在遊戲內體驗到「ScamAdviser 真的有用」，結算時被推薦的轉換率會更高；會診使用本身就是 funnel 指標 ③，可以直接衡量。

- **Q：為什麼只有 5 題？太短？**
  A：完測率優先於 scope。5 題涵蓋核心詐騙類型（AI 變聲、假投資、假物流、假求職、假公益）且控制在 3-5 分鐘內，保留玩家情緒高點進結算 + 分享環節。如果完測率超過 60%，下個版本可以加到 7 題。

- **Q：為什麼選擇急診醫師這個身份，不是臥底探員或詐騙受害者？**
  A：急診醫師有三個獨特優勢──(1) 能自然串接「診斷 / 處方 / 藥袋」整套輸出隱喻 (2) 沒有「被騙」的心理負擔 (3) 醫師有權威感 = 分享藥袋時看起來像「官方診斷書」，更有 credibility。
