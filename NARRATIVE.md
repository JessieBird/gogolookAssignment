# 🏥 人間清醒醫院 · 沈浸式劇情設計說明

> 這份文件記錄「為什麼這樣說故事」—— 從你是誰、Dr. 清醒 是誰、每個畫面在醫療世界觀裡對應什麼場景，到每個敘事決策背後的思考。
> 給自己研讀用，面試時用來 defend 設計決策。

---

## 🎭 核心敘事定位

### 使用者身份翻轉
**原始設計**：你是急診醫師，幫別人診斷
**最終設計**：你是病患，來做反詐體質健檢

### 為什麼翻轉？
原本有一個隱性矛盾：
- 身份設定 → 你是醫師
- 但**診斷書**是你的人格
- **處方箋**開給你
- **藥袋**裡的藥給你吃

病患視角一改，整條敘事邏輯對齊：**你來看診 → 你被診斷 → 你拿處方 → 你領藥**。
這樣做讓整個醫療隱喻從「假裝是誰」變成「真的在做健檢」。

### 主治醫師：Dr. 清醒
- 名字來自「人間清醒」的梗，保留品牌一致性
- 身兼：遊戲中的主治醫師 + 處方箋簽名醫師 + 藥袋上的調劑藥師
- 角色功能：作為 NPC 陪伴玩家做完整個流程，降低「自己跟系統對話」的冷感

## 📍 完整場景流程

### Scene 1 · 診所門口（Home）

**使用者看到什麼：**
- 紅色急診燈脈動 ICON
- 「人間清醒醫院」大標
- 「反詐體質門診」副標
- 文案：「最近可疑訊息收到手軟嗎？本院為您開辦『反詐體質健檢』，主治 Dr. 清醒」
- 「前往掛號處」按鈕

**設計意圖：**
- **開場要立刻告訴你這是醫院場景**（視覺元素 + 醫療命名）
- **降低防衛心**—「反詐體質健檢」聽起來像正當身體檢查，不是來考你智商
- **置入 Dr. 清醒**—讓角色在第一眼就認識到玩家
- **CTA 是「掛號」而非「開始測驗」**—保持醫療敘事

**使用者心境：**
「我需要看一下嗎？」→ 好奇點進去

### Scene 2 · 掛號櫃檯（Onboarding）

**使用者看到什麼：**
1. **badge**：`掛號櫃檯 · REGISTRATION`
2. **標題**：「初診掛號單」
3. **副標**：「請出示您的『全民防詐健康保險卡』」
4. **健保卡視覺**：
   - 綠色漸層卡面 + 兩個手牽手的浮水印（模仿真實健保卡 logo）
   - H logo 圓章（白底深綠 H）
   - 「全民防詐健康保險」 / NATIONAL ANTI-SCAM INSURANCE
   - 金色晶片 + 左側三角形指示
   - **姓名**（即時從下方 input 同步到卡上，28px 大字置中）
   - **身分證字號**：`A165165165`（向 165 反詐騙專線致敬）
   - **初診日期**：今日（民國年格式，如 115/04/19）
   - **🤒 大頭貼**（右側，病患狀態 emoji）
   - **卡號**：`0000 165 5678`
5. **初診問卷**：「收到可疑訊息，您第一反應通常是？」→ 4 選項（馬上查證 / 問別人意見 / 憑感覺判斷 / 先行動再想）
6. **送出按鈕**：「完成掛號，進入看診」

**設計意圖：**
- **台灣人最有共鳴的醫療場景入口**—健保卡 100% 在地辨識度
- **姓名即時上卡**→ 建立「這張卡真的是我的」所有權感
- **165 彩蛋**—ID 和卡號都嵌入反詐專線號碼，增加細節彩蛋與主題一致性
- **初診問卷用健檢口吻**，不是「請做心理測驗」的僵硬感
- **4 種初始反應**僅作為後續 Result 頁「自我認知對照」的前測資料，不影響計分

**使用者心境：**
「這看起來好正式，好像真的在掛號」→ 填完，送出

### Scene 3 · 診間看診（Game × 5 題）

**使用者看到什麼（Header）：**
```
🔴 人間清醒醫院 門診區 | 🩺 主治 · Dr. 清醒 | 🤒 jess [傾向] | ❤️❤️❤️ XP 0 1/5
```
- 左側：醫院 logo
- 中間：兩個 badges（白底主治 + 青綠病患）
- 右側：HUD（愛心 / XP / 題數）

**使用者看到什麼（主畫面 · 左側案例卡）：**
- 病患照片佔位 👤
- 病患姓名 + 年齡 + 職業（例：王大明 65 歲 退休廚師）
- 案例類型嚴重程度標籤（🟠重症 + 🚨極高風險）
- **【急訴情境】** 描述此案例
- **🩺 看看醫師怎麼說**：3 個可點開的線索 chips（通話紀錄、背景音分析、病患描述等）
- （重症題 only）**會診報告**：點擊會診按鈕後展開的橘色區塊

**使用者看到什麼（主畫面 · 右側選項區）：**
- **🤔 如果是您，會建議怎麼做？**
- 4 個選項按鈕
- （重症題 only）「🔍 呼叫 ScamAdviser 會診（25 XP）」按鈕

**核心機制：**

| 機制 | 規則 |
|---|---|
| **XP 累積** | `XP = 選項 dim 總和 × 5`。最佳選項 +30~40，中等 +15~25，最爛 +0 且**扣 1 愛心** |
| **愛心** | 開場 3 顆，選到最爛（被詐騙）選項扣 1；歸零觸發反詐觀察病房 |
| **會診** | 重症題才可用，花 25 XP 揭露關鍵線索 + 標記一個明顯錯誤選項（4→3 選 1） |
| **浮動回饋** | 選完 1.1 秒彈出「+XP」或「−愛心」動畫才進下一題 |

**5 題情境：**
1. AI 變聲詐騙（重症） — 王大明接到「車禍」電話要匯款
2. 假投資 LINE 群 — 林美玲被朋友拉進「AI 教父內部投資群」
3. 假物流簡訊 — 陳志豪收到「包裹地址有誤」簡訊
4. 假求職陷阱（重症） — 張雅婷被要求繳 3000 保證金
5. 假公益募款 — 吳宗恩看到 IG 網紅受災戶募款

**設計意圖：**
- **案例題目保留「第三人稱情境」** 讓這變成智力測驗／體質健檢的問卷，不是「你自己正在被騙」的焦慮感
- **「看看醫師怎麼說」把線索擬人化成 Dr. 清醒 的提示**，醫師角色真的在幫你思考
- **ScamAdviser 會診 = 遊戲內產品整合**，在玩家最需要幫助時主動使用它（不是結算後才推銷）
- **扣愛心 = 「你會被騙」** → 具象化風險，比單純扣分更戲劇化

**使用者心境：**
「這題我要仔細想一下」→ 查線索 → 選答案 → 看回饋 → 下一題

### Scene 4 · 觀察病房通報（Death Screen）

**觸發時機**：累積 3 次選到最爛選項 → 愛心歸零

**使用者看到什麼：**
- 💀 骷髏頭 icon（紅底白骷髏）
- **CODE RED · 防詐警報**
- 標題：**反詐觀察病房通知**
- 主文：「[你名字] 病患：您在本次健檢中已不幸『被詐騙』3 次，主治醫師 Dr. 清醒 建議您立即轉入反詐觀察病房留觀。」
- 療養建議框：「前往 ScamAdviser 翻閱最新詐騙案例庫，補強防詐知識後即可取得『出院許可證』，回到診間完成剩餘健檢。」
- 兩顆按鈕：
  - 🔴 **前往 ScamAdviser 補課出院**（外連）
  - ⚪ **接受留觀，繼續完成剩餘健檢題**（關 Modal 續玩）

**設計意圖：**
- **不是 game over**—敘事上是「留觀」不是「失敗」，病患可以繼續看診
- **黑色幽默降低挫折感**—「不幸被詐騙 3 次」是開玩笑語氣
- **CTA 在情緒最高點**—被詐 3 次的瞬間，玩家自然有「我該去學一下」的驅力
- **不強制外連** → 尊重玩家 agency，保留「不想去的人」也能繼續完賽

### Scene 5 · 醫師宣布診斷（Result 頁）

**使用者看到什麼（診斷書本體）：**
1. **標題**：「人間清醒醫院防詐人格診斷書 / STAY-WOKE REPORT」
2. **受測資料**：醫師姓名（你）/ 初始反應 / 人格類型標籤
3. **三維指數**：警覺指數 / 理性指數 / 防詐體質（S+ 到 D）
4. **最終診斷區塊**：
   - 人格名稱（6 種之一）
   - 段位（頂尖主任醫師 → 可憐實習生 → ...）
   - **診斷印章**（斜角圓章，例：「免疫出院」、「需要住院」、「需觀察」）
   - **臨床備註**：為該人格量身撰寫的冷嘲熱諷

**使用者看到什麼（自我認知對照區塊）：**
- 你宣稱：[初始反應] → 診斷結果：[人格]
- **24 組幽默文案**中挑一個（根據宣稱 × 診斷的組合）
- 例：宣稱「馬上查證」+ 診斷「善良受難者」→
  > 「你以為自己會馬上查證？知道跟做到之間，差了 5 通詐騙電話的距離。」

**使用者看到什麼（CTA 區）：**
- 🔴 **領取我的個人化處方箋 RX**（主 CTA + NEW 標）
- 🔵 **分享我的防詐人格**（次 CTA）
- 灰色小按鈕：**重新實習（再測一次）**

**6 種人格：**

| 人格 | 段位 | 印章 |
|---|---|---|
| 冷血分析師 | 頂尖主任醫師 | 免疫出院 |
| 善良受難者 | 可憐實習生 | 需要住院 |
| 嘴砲專家 | 資深主治醫師 | 需觀察 |
| 直覺獵人 | 急診總醫師 | 觀察中 |
| 制度信徒 | 住院醫師 | 需補修 |
| 混沌中立者 | 謎之醫師 | 特殊體質 |

**設計意圖：**
- **診斷書用正式醫療文件視覺**—建立「這是權威判決」的正式感
- **自我認知對照是核心分享引擎**—宣稱 vs 實際的落差最有故事性（「我以為我是 X，結果 Y」）
- **段位化人格設定**讓玩家有「升段」動機（下次再測可以更好）
- **三維指數用 GPA 等級 S+/A/B** → 普世可懂的成就感

- **連結選擇**：都導向 scamadviser.com 的相關頁面，不混用 Scam University（避免品牌分散）

## 💡 面試可被 challenge 的問題 & 回答

### Q1：為什麼選擇病患視角、不是醫師視角？
A：原本醫師視角有個隱性矛盾—你假裝幫別人看病，但**診斷書、處方、藥袋都是給你的**。改成病患視角後整條邏輯通順：「你來看診→你被診斷→你拿處方→你領藥」。這也讓整個醫療隱喻從 cosplay 變成真的在做健檢，代入感提升。

### Q2：為什麼不讓 5 題也改成第一人稱「你」遇到情境？
A：故意保留「第三人稱案例題」。理由：
- 病患視角下，這些像**智力測驗 / 體質健檢問卷**的題目，醫師用案例來量測你的判斷力
- 第一人稱會產生「我正在被騙」的焦慮感，反而降低參與意願
- 保留第三人稱讓你有**分析距離**（你在看別人的狀況，但你的選擇暴露了你的 pattern）

### Q3：為什麼要放健保卡？會不會太本地化？
A：**故意極度本地化**。這個 MVP 是針對台灣市場驗證假設，健保卡是 100% 在地符號—不用解釋、一看就懂、自帶親切感。如果驗證成功，未來推到其他地區再做對應的本地 onboarding（美國用 social security card、日本用 my number card）。本地化程度高正是社交擴散的燃料之一。

### Q4：Dr. 清醒 會不會太耍冷？
A：**故意耍冷**。Dr. 清醒 = 人間清醒 的醫療化 + 擬人化。品牌一致性優先於嚴肅感。而且在處方箋、藥袋都會再次看到他簽名，角色熟悉度自然增加，最後分享時玩家會覺得「這醫師也蠻可愛」。

### Q5：既然你是病患，為什麼選項還叫「建議怎麼做」？
A：敘事上，醫師在做一種**情境模擬測驗**—「如果你遇到這個朋友來問你，你會怎麼說？」你的答案暴露了你真實會怎麼反應。這個迂迴的設計其實跟心理學上的 projection 類似，人們在回答「別人該怎麼做」時往往比「你該怎麼做」更誠實。

### Q6：為什麼把「陣亡」改成「留觀」？
A：醫師視角的「陣亡」是他的病患死了（荒謬但有黑色幽默）。病患視角如果「陣亡」就是你死了—這太遠，且敘事上病患不會自己宣布自己死。「被詐騙 3 次送進觀察病房」更合理：仍然有戲劇性（進病房留觀），但不脫離病患在醫院裡的合理經驗。
